

# Knowledge Base

##  Быстрый старт

Для запуска проекта вам понадобятся:
* **Node.js** (версия 18+)
* **Python 3.x** (для работы скрипта синхронизации)
* **Git**

### 1. Установка зависимостей
Сначала установите необходимые пакеты для работы интерфейса:
```bash
npm install
```

### 2. Запуск и обновление
При запуске команды `dev`, проект автоматически проверит наличие свежих статей в репозитории [knowledgebse](https://github.com/svyatoylol/knowledgebse) и подтянет изменения:
```bash
npm run dev
```
> **Примечание:** Скрипт синхронизации использует `git sparse-checkout`, чтобы скачивать только `.md` файлы и изображения, экономя ваш трафик и место на диске.

---

##  Команды

| Команда | Описание |
| :--- | :--- |
| `npm run sync` | Только обновить статьи из облака (без запуска сайта). |
| `npm run dev` | Синхронизировать контент и запустить локальный сервер для чтения. |
| `npm run build` | Обновить контент и собрать проект в статичный HTML. |
