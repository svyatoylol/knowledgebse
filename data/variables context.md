# Контекст (область видимости) переменных в C#

Область видимости (**scope**) — это часть кода, в которой переменная **существует и доступна**.

## Базовое правило

Переменная живёт внутри блока `{ }`, в котором объявлена.

```csharp
{
    int x = 10;
    Console.WriteLine(x); // ✅ работает
}

Console.WriteLine(x); // ❌ ошибка — x здесь не существует
```

## Уровни области видимости

### Локальные переменные (внутри метода)

```csharp
void Greet()
{
    string message = "Привет!";
    Console.WriteLine(message); // ✅
}

// message здесь недоступна
```

### Переменные блока (if, for, while...)

```csharp
if (true)
{
    int number = 42;
    Console.WriteLine(number); // ✅
}

Console.WriteLine(number); // ❌ ошибка
```

```csharp
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i); // ✅
}

Console.WriteLine(i); // ❌ ошибка — i живёт только внутри for
```

### Поля класса (уровень класса)

```csharp
class Player
{
    string name = "Герой"; // доступна во всех методах класса
    int health = 100;

    void PrintInfo()
    {
        Console.WriteLine(name);   // ✅
        Console.WriteLine(health); // ✅
    }
}
```

## Вложенные блоки

Внутренний блок видит переменные внешнего, но не наоборот:

```csharp
int outer = 10;

{
    int inner = 20;
    Console.WriteLine(outer); // ✅ внутренний видит внешний
    Console.WriteLine(inner); // ✅
}

Console.WriteLine(outer); // ✅
Console.WriteLine(inner); // ❌ внешний не видит внутренний
```

## Конфликт имён

Нельзя объявить переменную с тем же именем во вложенном блоке, если она уже есть во внешнем:

```csharp
int x = 10;

{
    int x = 20; // ❌ ошибка — x уже объявлена выше
}
```

## Переменная не инициализирована

C# не даст использовать локальную переменную до присвоения значения:

```csharp
int number;
Console.WriteLine(number); // ❌ ошибка компиляции
```

```csharp
int number = 0;
Console.WriteLine(number); // ✅
```

## Итого

| Уровень | Где доступна |
|---|---|
| Блок `{ }` | только внутри этого блока |
| Метод | во всём методе |
| Поле класса | во всех методах класса |
| Вложенный блок | видит переменные родительского |
