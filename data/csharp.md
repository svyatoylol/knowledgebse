# Введение в C#: От основ до современных возможностей

C# (C-Sharp) — это мощный, объектно-ориентированный язык программирования, разработанный компанией Microsoft в начале 2000-х годов. С тех пор он прошел огромный путь, превратившись из "языка для Windows" в универсальный инструмент для создания кроссплатформенных приложений, игр на Unity, облачных сервисов и мобильных решений.

## Почему C# популярен?

1.  **Типобезопасность:** Язык минимизирует ошибки на этапе компиляции, проверяя типы данных.
2.  **Управление памятью:** Сборщик мусора (Garbage Collector) автоматически освобождает память, что снижает риск утечек.
3.  **Огромная экосистема:** Платформа .NET предоставляет тысячи готовых библиотек (через NuGet).
4.  **Кроссплатформенность:** С выходом .NET Core (а теперь просто .NET 5/6/7/8) код на C# работает на Windows, Linux и macOS.
