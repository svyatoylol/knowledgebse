# Цикл while в C#

## Что такое цикл

Цикл — это конструкция, которая позволяет выполнять один и тот же код несколько раз.

Без циклов пришлось бы писать одинаковые команды вручную.

Например, вместо:

```csharp
Console.WriteLine(1);
Console.WriteLine(2);
Console.WriteLine(3);
Console.WriteLine(4);
Console.WriteLine(5);
````

можно использовать цикл.

---

# Что такое while

`while` повторяет код, пока условие истинно (`true`).

---

# Синтаксис while

```csharp
while (условие)
{
    код;
}
```

---

# Как работает while

1. Проверяется условие.
2. Если условие истинно — выполняется код внутри цикла.
3. После выполнения программа снова проверяет условие.
4. Цикл повторяется, пока условие остаётся истинным.

---

# Простейший пример

```csharp
int i = 1;

while (i <= 5)
{
    Console.WriteLine(i);

    i++;
}
```

---

# Разбор программы

## Переменная-счётчик

```csharp
int i = 1;
```

Создаётся переменная `i`.

---

## Условие цикла

```csharp
i <= 5
```

Цикл работает, пока `i` меньше или равно `5`.

---

## Изменение счётчика

```csharp
i++;
```

`++` увеличивает значение на `1`.

Это сокращение для:

```csharp
i = i + 1;
```

---

# Что выведет программа

```text
1
2
3
4
5
```

---

# Цикл с вводом данных

Пример программы, которая работает, пока пользователь не введёт `"exit"`.

```csharp
string command = "";

while (command != "exit")
{
    Console.Write("Введите команду: ");

    command = Console.ReadLine();
}
```

# Бесконечный цикл

Если условие всегда истинно, цикл никогда не завершится.

Пример:

```csharp
while (true)
{
    Console.WriteLine("Бесконечный цикл");
}
```

---

# Почему возникают бесконечные циклы

Часто причина — забыли изменить переменную.

Неправильно:

```csharp
int i = 1;

while (i <= 5)
{
    Console.WriteLine(i);
}
```

Здесь `i` никогда не изменяется.

Условие всегда остаётся истинным.

---

# Цикл с bool

Условие должно возвращать `true` или `false`.

Пример:

```csharp
bool isRunning = true;

while (isRunning)
{
    Console.WriteLine("Программа работает");

    isRunning = false;
}
```

---

# Использование break

`break` полностью завершает цикл.

```csharp
while (true)
{
    string text = Console.ReadLine();

    if (text == "stop")
    {
        break;
    }
}
```

---

# Использование continue

`continue` пропускает оставшийся код текущей итерации.

```csharp
int i = 0;

while (i < 5)
{
    i++;

    if (i == 3)
    {
        continue;
    }

    Console.WriteLine(i);
}
```

---

# Вложенные циклы while

Один цикл можно помещать внутрь другого.

```csharp
int i = 1;

while (i <= 3)
{
    int j = 1;

    while (j <= 2)
    {
        Console.WriteLine(i + " " + j);

        j++;
    }

    i++;
}
```

---

# Цикл while и do while

`while` сначала проверяет условие.

`do while` сначала выполняет код.

## while

```csharp
while (false)
{
    Console.WriteLine("Текст");
}
```

Код не выполнится ни разу.

---

## do while

```csharp
do
{
    Console.WriteLine("Текст");
}
while (false);
```

Код выполнится один раз.

---

# Частые ошибки

## Забыли изменить счётчик

```csharp
while (i < 10)
{
    Console.WriteLine(i);
}
```

Цикл станет бесконечным.

---

# Ошибка в условии

```csharp
while (i = 5)
```

Неправильно.

Нужно:

```csharp
while (i == 5)
```

---

# Пример программы

Программа считает сумму чисел от 1 до 5.

```csharp
int i = 1;
int sum = 0;

while (i <= 5)
{
    sum += i;

    i++;
}

Console.WriteLine(sum);
```

---

# Что такое итерация

Одно выполнение тела цикла называется итерацией.

В примере:

```csharp
while (i <= 5)
```

цикл выполнится 5 итераций.
