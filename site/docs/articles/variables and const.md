# Переменные и константы в C#

## Что такое переменная

Переменная — это область памяти, в которой хранится значение.

Программа может:
- сохранять данные в переменные;
- изменять их;
- использовать в вычислениях.

Пример переменной:

```csharp
int age = 18;
````

Здесь:

* `int` — тип данных;
* `age` — имя переменной;
* `18` — значение переменной.

---

# Зачем нужны переменные

Без переменных программа не смогла бы хранить:

* числа;
* текст;
* результаты вычислений;
* данные пользователя.

Например:

```csharp
string name = "Анна";

Console.WriteLine(name);
```

Программа сохраняет имя в переменную и затем выводит его на экран.

---

# Объявление переменной

Создание переменной называется объявлением.

Общий вид:

```csharp
тип имя = значение;
```

Пример:

```csharp
double price = 99.99;
```

---

# Основные типы переменных

## int — целые числа

```csharp
int count = 10;
```

Хранит целые числа:

* `1`
* `25`
* `-100`

---

## double — дробные числа

```csharp
double weight = 72.5;
```

Хранит числа с дробной частью.

---

## string — строки

```csharp
string city = "Москва";
```

Хранит текст.

Строки записываются в кавычках.

---

## char — символ

```csharp
char letter = 'A';
```

Хранит один символ.

Символы записываются в одинарных кавычках.

---

## bool — логический тип

```csharp
bool isOnline = true;
```

Может хранить только:

* `true`
* `false`

---

# Изменение значения переменной

Значение переменной можно менять.

```csharp
int score = 10;

score = 20;
```

Теперь в `score` хранится `20`.

---

# Использование переменных

Переменные можно использовать в вычислениях.

```csharp
int a = 5;
int b = 3;

int sum = a + b;

Console.WriteLine(sum);
```

---

# Инициализация переменной

Инициализация — это присвоение первого значения.

```csharp
int age = 18;
```

Можно объявить переменную отдельно:

```csharp
int age;

age = 18;
```

Но использовать переменную до присвоения значения нельзя.

---

# Имена переменных

## Правила именования

Имя переменной:

* может содержать буквы и цифры;
* не может начинаться с цифры;
* не должно содержать пробелы;
* не должно совпадать с ключевыми словами языка.

Правильно:

```csharp
int userAge;
string firstName;
```

Неправильно:

```csharp
int 1age;
string user name;
```

---

# Стиль именования

В C# обычно используется стиль `camelCase`.

Первая буква маленькая, остальные слова — с большой буквы.

Примеры:

```csharp
firstName
userAge
productPrice
```

---

# Ключевое слово var

Иногда тип можно не писать явно.

```csharp
var age = 18;
```

C# сам определит тип переменной.

В этом примере `age` станет `int`.

Ещё пример:

```csharp
var name = "Alex";
```

Здесь тип будет `string`.

---

# Что такое константа

Константа — это значение, которое нельзя изменить после создания.

Для констант используется `const`.

```csharp
const double Pi = 3.14;
```

---

# Особенности констант

Константа:

* должна получить значение сразу;
* не может быть изменена позже.

Неправильно:

```csharp
const int MaxValue;

MaxValue = 100;
```

Правильно:

```csharp
const int MaxValue = 100;
```

---

# Попытка изменить константу

Такой код вызовет ошибку:

```csharp
const int Days = 7;

Days = 10;
```

Константы неизменяемы.

---

# Когда использовать константы

Константы подходят для значений, которые не должны меняться.

Например:

* число дней в неделе;
* число месяцев;
* математические значения;
* настройки программы.

Пример:

```csharp
const int MonthsInYear = 12;
```

---

# Разница между переменной и константой

| Переменная                      | Константа                            |
| ------------------------------- | ------------------------------------ |
| Значение можно изменить         | Значение изменить нельзя             |
| Используется для обычных данных | Используется для постоянных значений |
| Создаётся без `const`           | Создаётся с `const`                  |

---

# Пример программы

```csharp
const double Pi = 3.14;

double radius = 5;

double area = Pi * radius * radius;

Console.WriteLine(area);
```

Программа вычисляет площадь круга.
