# Цикл for в C#

## Что такое цикл

Цикл позволяет выполнять один и тот же код несколько раз.

Цикл `for` чаще всего используется тогда, когда количество повторений известно заранее.

Например:
- вывести числа от 1 до 10;
- пройти по элементам массива;
- выполнить действие несколько раз.

---

# Синтаксис цикла for

```csharp 
for (начало; условие; изменение)
{
    код;
}
````

---

# Простейший пример

```csharp id="h5m8w1"
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}
```

---

# Как работает цикл for

Цикл состоит из трёх частей.

## 1. Начало

```csharp 
int i = 1
```

Создаётся переменная-счётчик.

---

## 2. Условие

```csharp 
i <= 5
```

Цикл работает, пока условие истинно.

---

## 3. Изменение

```csharp 
```

После каждой итерации значение `i` увеличивается на `1`.

---

# Что выведет программа

```text 
1
2
3
4
5
```

---

# Как выполняется цикл

Для кода:

```csharp 
for (int i = 1; i <= 3; i++)
{
    Console.WriteLine(i);
}
```

происходит следующее:

1. `i = 1`
2. Проверка `i <= 3`
3. Выполнение кода
4. `i++`
5. Новая проверка условия

---

# Использование i--

Счётчик можно уменьшать.

```csharp
for (int i = 5; i >= 1; i--)
{
    Console.WriteLine(i);
}
```

---

# Шаг цикла

Счётчик можно изменять не только на `1`.

## Увеличение на 2

```csharp 
for (int i = 0; i <= 10; i += 2)
{
    Console.WriteLine(i);
}
```

---

# Блок кода

Тело цикла записывается в `{ }`.

```csharp id="m6w8xp"
for (int i = 1; i <= 3; i++)
{
    Console.WriteLine("Начало");

    Console.WriteLine(i);

    Console.WriteLine("Конец");
}
```

---

# Использование break

`break` полностью завершает цикл.

```csharp id="j4x2ka"
for (int i = 1; i <= 10; i++)
{
    if (i == 5)
    {
        break;
    }

    Console.WriteLine(i);
}
```

---

# Использование continue

`continue` пропускает текущую итерацию.

```csharp id="d9t1oy"
for (int i = 1; i <= 5; i++)
{
    if (i == 3)
    {
        continue;
    }

    Console.WriteLine(i);
}
```

---

# Вложенные циклы

Один цикл можно помещать внутрь другого.

```csharp 
for (int i = 1; i <= 3; i++)
{
    for (int j = 1; j <= 2; j++)
    {
        Console.WriteLine(i + " " + j);
    }
}
```

---

# Бесконечный цикл for

Если условие всегда истинно, цикл станет бесконечным.

```csharp 
for (;;)
{
    Console.WriteLine("Бесконечный цикл");
}
```

# Пример вычисления суммы

Программа считает сумму чисел от 1 до 5.

```csharp 
int sum = 0;

for (int i = 1; i <= 5; i++)
{
    sum += i;
}

Console.WriteLine(sum);
```

---

# Использование for со строками

Можно выводить символы строки по одному.

```csharp 
string text = "Hello";

for (int i = 0; i < text.Length; i++)
{
    Console.WriteLine(text[i]);
}
```

# Отличие for от while

## for

Используется, когда количество повторений известно.

```csharp 
for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}
```

---

## while

Используется, когда повторение зависит от условия.

```csharp 
while (command != "exit")
{
    command = Console.ReadLine();
}
```

---

# Частые ошибки

## Забыли изменить счётчик

```csharp 
for (int i = 0; i < 5;)
{
    Console.WriteLine(i);
}
```

Счётчик не изменяется.

Цикл станет бесконечным.

---

# Ошибка в условии

```csharp 
for (int i = 0; i = 5; i++)
```

Неправильно.

Нужно:

```csharp
for (int i = 0; i == 5; i++)
```

или другое корректное условие.

---

# Что такое итерация

Одно выполнение тела цикла называется итерацией.

В примере:

```csharp 
for (int i = 1; i <= 5; i++)
```

будет выполнено 5 итераций.
