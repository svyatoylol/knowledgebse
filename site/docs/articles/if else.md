# Условные конструкции if и else в C#

## Что такое условие

Условие позволяет программе принимать решения.

Например:
- если пользователь ввёл правильный пароль — разрешить вход;
- если число больше нуля — вывести сообщение;
- если возраст меньше 18 — запретить доступ.

В C# для этого используются:
- `if`
- `else`
- `else if`

---

# Конструкция if

`if` выполняет код только тогда, когда условие истинно (`true`).

## Синтаксис

```csharp
if (условие)
{
    код;
}
```

---

# Простейший пример

```csharp
int age = 20;

if (age >= 18)
{
    Console.WriteLine("Доступ разрешён");
}
```

---

# Как работает условие

В примере:

```csharp
age >= 18
```

оператор `>=` проверяет:

> возраст больше или равен 18?

Если условие истинно, код внутри `{ }` выполняется.

---

# Операторы сравнения

## Равно

```csharp
x == y
```

Проверяет, равны ли значения.

---

## Не равно

```csharp
x != y
```

---

## Больше

```csharp
x > y
```

---

## Меньше

```csharp
x < y
```

---

## Больше или равно

```csharp
x >= y
```

---

## Меньше или равно

```csharp
x <= y
```

---

# Блок кода в if

Фигурные скобки содержат команды, которые выполняются при истинном условии.

```csharp
int number = 10;

if (number > 0)
{
    Console.WriteLine("Число положительное");
    Console.WriteLine("Проверка завершена");
}
```

---

# Конструкция else

`else` выполняется, если условие в `if` ложно.

## Синтаксис

```csharp
if (условие)
{
    код 1;
}
else
{
    код 2;
}
```

---

# Пример if else

```csharp
int age = 15;

if (age >= 18)
{
    Console.WriteLine("Доступ разрешён");
}
else
{
    Console.WriteLine("Доступ запрещён");
}
```

---

# Конструкция else if

`else if` позволяет проверять несколько условий подряд.

## Пример

```csharp
int score = 75;

if (score >= 90)
{
    Console.WriteLine("Отлично");
}
else if (score >= 70)
{
    Console.WriteLine("Хорошо");
}
else
{
    Console.WriteLine("Нужно улучшить результат");
}
```

---

# Порядок проверки условий

Условия проверяются сверху вниз.

Как только найдено истинное условие:
- остальные проверки пропускаются.

---

# Логический тип bool

Условия работают с типом `bool`.

`bool` может хранить:
- `true`
- `false`

Пример:

```csharp
bool isOnline = true;

if (isOnline)
{
    Console.WriteLine("Пользователь в сети");
}
```

---

# Логические операторы

## Оператор И — &&

Условие истинно, только если истинны обе части.

```csharp
int age = 20;
bool hasTicket = true;

if (age >= 18 && hasTicket)
{
    Console.WriteLine("Вход разрешён");
}
```

---

# Оператор ИЛИ — ||

Достаточно, чтобы хотя бы одно условие было истинным.

```csharp
if (day == "Суббота" || day == "Воскресенье")
{
    Console.WriteLine("Выходной");
}
```

---

# Оператор НЕ — !

Меняет значение на противоположное.

```csharp
bool isBlocked = false;

if (!isBlocked)
{
    Console.WriteLine("Доступ открыт");
}
```

---

# Вложенные if

Один `if` можно помещать внутрь другого.

```csharp
int age = 20;
bool hasTicket = true;

if (age >= 18)
{
    if (hasTicket)
    {
        Console.WriteLine("Проходите");
    }
}
```

---

# Сокращённая запись

Если внутри `if` только одна инструкция, фигурные скобки можно не писать.

```csharp
if (x > 0)
    Console.WriteLine("Положительное число");
```

Но для начинающих лучше всегда использовать `{ }`.

---

# Частые ошибки

## Использование = вместо ==

Неправильно:

```csharp
if (x = 5)
```

Правильно:

```csharp
if (x == 5)
```

`=` — присваивание,  
`==` — сравнение.

---

# Ошибка с точкой с запятой

Неправильно:

```csharp
if (x > 0);
{
    Console.WriteLine(x);
}
```

Из-за `;` условие перестаёт работать правильно.
