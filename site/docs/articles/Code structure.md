# Базовый синтаксис C#

## Что такое синтаксис

Синтаксис — это набор правил, по которым пишется код на языке программирования.  
Если нарушить правила синтаксиса, программа не сможет скомпилироваться.

В C# код состоит из инструкций, блоков кода, переменных, методов и других конструкций.

---

# Структура самой простой программы

Простейшая программа на C# выглядит так:

```csharp
Console.WriteLine("Привет, мир!");
````

Эта строка выводит текст в консоль.

---

# Инструкции в C#

Инструкция — это отдельная команда программы.

Пример:

```csharp
Console.WriteLine("Первая строка");
Console.WriteLine("Вторая строка");
```

Каждая инструкция в C# обычно заканчивается точкой с запятой `;`.

Если забыть `;`, компилятор выдаст ошибку.

---

# Блоки кода

Блок кода — это набор инструкций внутри фигурных скобок `{ }`.

Пример:

```csharp
{
    Console.WriteLine("Команда 1");
    Console.WriteLine("Команда 2");
}
```

Фигурные скобки используются почти везде:

* в методах;
* условиях;
* циклах;
* классах.

# Комментарии

Комментарии нужны для пояснения кода.
Компилятор их игнорирует.

## Однострочный комментарий

```csharp
// Это комментарий
Console.WriteLine("Hello");
```

## Многострочный комментарий

```csharp
/*
   Это
   многострочный
   комментарий
*/
```


# Вывод данных в консоль

Для вывода используется `Console.WriteLine`.

## Вывод текста

```csharp
Console.WriteLine("Привет");
```

## Вывод переменной

```csharp
string name = "Анна";

Console.WriteLine(name);
```

## Вывод нескольких значений

```csharp
string name = "Анна";
int age = 20;

Console.WriteLine(name + " " + age);
```

# Ввод данных

Для чтения текста используется `Console.ReadLine`.

```csharp
string name = Console.ReadLine();
```

Пример программы:

```csharp
Console.Write("Введите имя: ");

string name = Console.ReadLine();

Console.WriteLine("Привет, " + name);
```

# Методы

Метод — это набор команд с именем.

```csharp
void SayHello()
{
    Console.WriteLine("Привет");
}
```

Вызов метода:

```csharp
SayHello();
```

# Форматирование кода

В C# принято использовать отступы и переносы строк.

Хороший стиль:

```csharp
if (x > 0)
{
    Console.WriteLine(x);
}
```

Плохой стиль:

```csharp
if(x>0){Console.WriteLine(x);}
```
