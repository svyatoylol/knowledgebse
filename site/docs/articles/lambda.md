# Лямбда-выражения в C#

Лямбда — это **короткая анонимная функция**, которую можно записать прямо в месте использования.

## Синтаксис

```csharp
(параметры) => выражение
(параметры) => { блок кода }
```

## Простые примеры

```csharp
// Одна строка — без фигурных скобок и return
Func<int, int> square = x => x * x;
Console.WriteLine(square(5)); // 25

// Несколько параметров
Func<int, int, int> add = (a, b) => a + b;
Console.WriteLine(add(3, 4)); // 7

// Без параметров
Action greet = () => Console.WriteLine("Привет!");
greet(); // Привет!

// Несколько строк — нужны скобки и return
Func<int, int> abs = x =>
{
    if (x < 0) return -x;
    return x;
};
```

## Типы параметров

Обычно компилятор выводит типы сам, но можно указать явно:

```csharp
Func<int, int> square = (int x) => x * x;
```

## Лямбда как аргумент метода

Лямбды чаще всего передают прямо в вызов — без сохранения в переменную:

```csharp
List<int> numbers = { 1, 2, 3, 4, 5, 6 };

// Найти все чётные
List<int> evens = numbers.Where(n => n % 2 == 0).ToList();

// Умножить каждый на 2
List<int> doubled = numbers.Select(n => n * 2).ToList();

// Отсортировать по убыванию
numbers.Sort((a, b) => b - a);
```

## Захват переменных (closure)

Лямбда может использовать переменные из окружающего кода:

```csharp
int multiplier = 3;
Func<int, int> multiply = x => x * multiplier;

Console.WriteLine(multiply(5)); // 15

multiplier = 10;
Console.WriteLine(multiply(5)); // 50 — лямбда видит актуальное значение!
```

Лямбда захватывает **ссылку** на переменную, а не её копию.

## Лямбда vs обычный метод

```csharp
// Обычный метод
int Square(int x) => x * x;

// То же самое — лямбда
Func<int, int> square = x => x * x;
```

Лямбды удобны, когда логика простая и нужна **только в одном месте**.

## Итого

| Что | Синтаксис |
|---|---|
| Один параметр | `x => x * x` |
| Несколько параметров | `(a, b) => a + b` |
| Без параметров | `() => Console.WriteLine("!")` |
| Несколько строк | `x => { ...; return ...; }` |
| Захват переменной | используется напрямую из scope |
