# C# и .NET: Полное руководство

<article>

## Введение в C# и платформу .NET

**C#** (произносится "си шарп") — современный, объектно-ориентированный язык программирования, разработанный Microsoft в рамках платформы **.NET**.

### Что такое .NET?
.NET — это бесплатная, открытая кроссплатформенная платформа для создания приложений любого типа:
- **Консольные приложения**
- **Веб-приложения** (ASP.NET Core)
- **Мобильные приложения** (.NET MAUI)
- **Игры** (Unity)
- **Микросервисы и облачные решения**

### Ключевые компоненты .NET:
```
┌─────────────────────────────────┐
│   Приложение (ваш код на C#)    │
├─────────────────────────────────┤
│   Common Language Runtime (CLR) │
│   • Управление памятью (GC)     │
│   • Безопасность типов          │
│   • Обработка исключений        │
├─────────────────────────────────┤
│   Базовая библиотека классов    │
│   (BCL - Base Class Library)    │
└─────────────────────────────────┘
```

### Первая программа на C#:
```csharp
// Program.cs
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Привет, мир!");
            Console.WriteLine("Нажмите любую клавишу для выхода...");
            Console.ReadKey();
        }
    }
}
```

### Структура проекта .NET:
```bash
my-app/
├── Program.cs          # Точка входа
├── my-app.csproj       # Файл проекта (MSBuild)
├── bin/                # Скомпилированные файлы
├── obj/                # Промежуточные файлы сборки
└── packages/           # NuGet-пакеты
```

### Компиляция и запуск:
```bash
# Создание нового консольного проекта
dotnet new console -n MyApp

# Переход в директорию
cd MyApp

# Запуск приложения
dotnet run

# Сборка проекта
dotnet build

# Публикация
dotnet publish -c Release -r win-x64
```

### Особенности C#:
- **Строгая типизация** с выводом типов (`var`)
- **Автоматическое управление памятью** (Garbage Collector)
- **Поддержка ООП**: наследование, полиморфизм, инкапсуляция
- **LINQ** для работы с данными
- **Асинхронность** через `async/await`
- **Кроссплатформенность** (.NET 5+)
